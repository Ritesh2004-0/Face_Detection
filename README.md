# Face_Detection
Real-time Facial Emotion Detection using OpenCV and Deep Learning This project detects human facial emotions in real time using a webcam. It uses OpenCV for face detection and a pre-trained CNN model built with TensorFlow/Keras to classify emotions such as Happy, Sad, Angry, Surprise, and more.
Real-Time Facial Emotion Detection

This project is a real-time facial emotion detection system built using Python, OpenCV, and Deep Learning (TensorFlow/Keras).
It detects a human face from a webcam and predicts the emotion displayed on the face.
